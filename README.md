# Python Subject Registration System

## Description

This project demonstrates how to use Python sets to register student subjects. It shows how sets automatically prevent duplicate entries and helps beginners understand when sets are useful.

## Features

- Store subjects using a set
- Register subjects
- Prevent duplicate entries
- Display registered subjects
- Count unique subjects

## Technologies Used

- Python 3

## Concepts Practiced

- Sets
- Loops
- User Input
- Variables
- Set Methods (`add()`, `remove()`)

## Future Improvements

- Validate subject names
- Save registrations to a file
- Register multiple students
- Display available subjects before registration
